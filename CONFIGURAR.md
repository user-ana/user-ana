# Configurar tu perfil bonito de GitHub

## 1. Que contiene esta carpeta

```text
github-profile-design/
├── README.md
└── assets/
    ├── profile-banner.svg
    └── terminal-card.svg
```

Este es el diseño para tu perfil de GitHub. El `README.md` usa los SVG de la carpeta `assets/`, asi que cuando lo subamos a GitHub hay que subir todo junto.

## 2. Cambios necesarios

En `README.md` cambia:

- `Ana Montes`, si quieres cambiar como aparece tu nombre.
- `user-ana`, si cambias tu usuario de GitHub.
- `bcoin5992@gmail.com`, si cambias tu correo.
- `cyrk_anag`, si cambias tu Instagram.

En `assets/profile-banner.svg` cambia:

- `ANA MONTES`

En `assets/terminal-card.svg` cambia:

- `Ana Montes`

## 3. Repositorio correcto

Para que GitHub muestre este README en tu perfil, el repositorio debe llamarse exactamente igual que tu usuario.

Ejemplo:

```text
Usuario: ana-dev
Repositorio: ana-dev
```

## 4. Estilo del perfil

Este diseno incluye:

- Banner animado oscuro.
- Animacion tipo escritura.
- Tarjeta estilo terminal animada.
- Stack de tecnologias.
- Proyecto VIP destacado.
- Estadisticas de GitHub.
- Seccion tipo CV.
- Contacto profesional.
